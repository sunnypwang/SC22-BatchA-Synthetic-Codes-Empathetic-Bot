jAIce

the empathetic robot here to comfort you.

Purpose

The purpose of jAIce is to make sure you have someone to go to for some comforting responses. This is a safe coping mechanism for the user since this allows positive venting to be taken place which can be a helpful method for people to process their emotions and reduce their stress; It can be used as a tool to aid with one’s mental well-being.

Application

While the software may not be applicable to professional therapeutic processes at this stage, it can be used as a tool to aid with one’s mental wellbeing. A large component of this coping method is that the receiver of such venting listens carefully and provides positive affirmation and/or comfort, which the software aims to do.

For further expansion, we may aim to provide practical advice, expand into a chatbot, or provide other features that are more well-suited for psychological treatment.

Description

jAIce uses a GPT-2 model developed by OpenAI. The dataset used for training is the Empathetic Dialogues dataset outlined in this paper. We cleaned the dataset by removing all questions and removing unnecessary data and quotations. We used Python and aitextgen to help train the model.

There are two parameters you can adjust. Higher temperature causes the model to come up with more specific response, while top_p controls how much the model is confident in its response (higher number is a higher percent of confidence).
